# Оригинальные Java-апплеты часов (1997)

Автор: iLya (Илья Осипов).

Здесь хранятся **оригинальные** скомпилированные апплеты, которыми на сайте
делались часы в конце 1990-х:

- `Clock.class` — аналоговые часы: картинка-циферблат + стрелки с тенью.
- `Clockconkit.class` — интерактивный конструктор часов («оболочка для
  разработки часов», страница `c_conkit.htm`).

## Статус

Эти файлы **не работают** в современных браузерах: поддержка Java-апплетов
(тег `<applet>` и Java-плагин) была убрана из всех браузеров примерно к 2017
году. Исходный код `Clock.java` / `Clockconkit.java` не сохранился — уцелел
только байт-код (эти `.class`).

Файлы оставлены **как исторический артефакт**. Запускать их не нужно и негде.

## Что показывается на сайте вместо них

Живые часы на страницах рисует `../clock-canvas.js` — это **новая**
реализация на чистом JavaScript (`<canvas>`), восстановленная автоматически
по строкам и именам методов внутри `Clock.class` (`image=`, `handcolor=`,
`shadowcolor=`, `bgcolor=`, `textcolor=`, `textsize=`, `TimeOffset=`,
`getHours/getMinutes/getSeconds`, `drawImage`, `drawLine`). Она читает ту же
строку параметров, что и оригинальный апплет.

Оригинальные теги `<applet>` на страницах не удалены, а закомментированы с
пометкой «ОРИГИНАЛ 1997» — чтобы исходная разметка тоже сохранилась.

Год — по штампам на самих страницах часов: «Created 24'09.97» и копирайту
«(C),97» (т.е. сентябрь 1997). Более ранней даты в сохранившемся архиве нет.

Оригинальный загрузчик-генератор апплета — `../clock.js` (тоже сохранён).
