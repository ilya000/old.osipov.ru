# Оригиналы HTML (1997–1998), без изменений

Здесь лежат **неизменённые оригиналы** страниц сайта — такими, какими они были
восстановлены из архива/Wayback, до правок для современного хостинга.

Отличия от живых страниц в корне (`../`):
- **кодировка windows-1251** (оригинальная), а не UTF-8;
- **нет** тега `<meta charset>`;
- **живые теги `<applet>`** Java-часов (Clock.class / Clockconkit.class) — не
  закомментированы, без `<canvas>`-замены;
- ссылки на картинки рекламных сетей (`counter.rambler.ru`, `linkexchange.ru`) —
  оригинальные внешние (ныне мёртвые).

Это исторический эталон для сверки. На живом сайте на эти файлы есть ссылки в
комментариях каждой страницы (`<!-- … оригинал … в original-html/… -->`).

Важно: файлы в windows-1251, а GitHub Pages отдаёт их с заголовком `charset=utf-8`,
поэтому при прямом открытии в браузере кириллица будет «кракозяблить» — это
нормально, они тут как байт-в-байт артефакт, а не для просмотра. Рабочие
(UTF-8) версии — в корне сайта.

`clock2.htm` и `clock3.htm` взяты из копии `www.ilya.nn.ru` (в Codex-реставрации
old.osipov.ru их не было); остальные 56 — из реставрации old.osipov.ru.
