Синтаксис часов.

Вернуться на главную страницу часов.

<APPLET CODE=Clock.class codebase="http://www.ilya.nn.ru/" Width=54 Height=72>
<PARAM NAME="clock" VALUE="image=round1.gif shadowcolor=#888877 bgcolor=#EEEEdd textcolor=#880088 handcolor=#FFFFFF textsize=12">
Java not found. </APPLET>

Где под (PARAM NAME="clock" VALUE=) - пишется строка параметров, которая может содержать следующие ключевые слова:

Параметры: *
  image=xxxxxx.gif - картинка используемая в качестве циферблата
  bgcolor=#RRGGBB - цвет фона часов, где RR,GG,BB - шеснадцатиричные значения красного,зеленого и синей компоненты цвета
  handcolor=#RRGGBB - цвет часовых стрелок.
  textcolor=#RRGGBB - цвет надписей.
  shadow - если тексту нужна тень.
  shadowcolor=#RRGGBB - цвет тени.
  textsize=xx - высота текстовых надписей, где xx - десятичное значение в диапазоне 5..25.
  calc - отладочный параметр, рассчитывающий Width и Height для этого апплета.
  timeoffset=xxx - Подстройка времени. (параметр добавлен по многочисленным просьбам) где ххх - относительное время в минутах, на которое нужно подправить часы. Например timeoffset=180 или timeoffset=-60

* Любой из параметров можно опустить.

Как с этим работать? - Вам просто нужно переместить текст помещенный в рамку, в текст вашей страницы, попутно настроив параметры под ваш стиль.

Существует только одно серьезное ограничение: циферблатные картинки можно использовать только из моей коллекции.
На сегодня доступны:

  view.gif
  sunset.gif

  globe.gif

  rol.gif
Если же вас не устраивает ни одна из перечисленных нарисуйте свою и пошлите мне. Если она будет интересна я обязательно добавлю ее в коллекцию и ссылку на вас.
