Hook Dump 2.7



The program-spy Hook Dump 2.7 writes to a file everything that was typed from the keyboard or clicked by the mouse while running in the computer background.

It is a perfect tool for spying on your friends, enemies, collegues or rivals. It lets you see everything that was entered from the keyboard including passwords. You can see what programs have been used, and how they've been used.

Here's how the program works... Hook Dump runs in the computer background, i.e. it is invisible for the user. It writes all the actions and whatever was entered from the keyboard to a file.

File is created automatically in the directory with a unique or specified name. The directory can be specified by editing HookDump.ini file. To enter a directory follow these basic steps:

- find the line that starts with [StartUp]
- put the next line right under it specifying what directory you want the file to be written to.

Example,

[StartUp]
Dir=x:\xxx\

By default, the file is written to a temp directory. You may also set a file name by typing

File=xxx.hk

If the filename hasn't been specified, the program will create a unique filename with extension .hk .

The recording of the user's actions starts immediately after the OS is started.

To remove Hook Dump from your system you'll need to restart the program in visible mode by entering a parameter /V. Now you can safely remove Hook Dump from your computer.
